# ™ 𝑨𝑯𝑴𝑨𝑫 𝑴𝑰𝑵𝑰 ᥫᩣ — Pair Website

Beautiful, ultra-fast pairing website for MINI AHMAD WhatsApp Bot.

## 🚀 Deploy on Vercel (2 Steps)

### Step 1: Connect to Vercel
1. Go to [vercel.com](https://vercel.com)
2. Click **"Add New Project"**
3. Import this GitHub repo
4. Framework Preset → **Other**
5. Click **Deploy** ✅

### Step 2: Configure
- Open your Vercel site
- Enter your bot server URL (Railway/Render/Koyeb) in the top input
- URL saves automatically in your browser!

## 🔐 Admin Panel (Hidden)

**Open Admin Panel:**
- Tap the logo/owner picture **5 times** quickly
- OR press `Ctrl + Shift + A`

**Admin Key:** Use the same key from your bot's `config.js` (`ADMIN_PANEL_KEY`)

### What You Can Control:
| Setting | Description |
|---------|-------------|
| 🎨 **Colors** | Primary, Accent, Accent 2 — live preview |
| 📸 **Owner Picture** | Show your photo above bot name |
| 🎵 **Background Music** | Auto-play music URL |
| 💬 **Messages** | Welcome msg, hint text, button text |
| 🔗 **Links** | WhatsApp, YouTube, Telegram, GitHub |
| 🆕 **Custom HTML** | Add ANY HTML — banners, ads, features |
| 📝 **Bot Name** | Change branding instantly |
| 🗑 **Reset** | One click to restore defaults |

## ✨ Features

- ⚡ No country restrictions — any number can pair
- 📱 Fully responsive (mobile + desktop)
- 🎨 Dynamic color themes
- 💾 Auto-saves settings (localStorage)
- 🎵 Background music toggle
- 🔒 Hidden admin panel
- 🆕 Custom HTML injection for new features

## 🔧 Custom Features via Admin

In the admin panel, use **"Extra HTML"** field to add:
- Announcement banners
- Custom links/buttons
- Embedded videos
- Social media widgets
- Any HTML you want!
